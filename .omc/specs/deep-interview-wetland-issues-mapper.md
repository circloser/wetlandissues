## Goal
국립생태원 습지센터 직원(수십~수백 명)이 **로그인 없이** 접속하여, **매일 자동 수집된 습지 관련 뉴스**가 습지센터 보유 습지 목록의 위치에 맵핑된 **전국 지도**를 보고, 이슈를 **하나씩 점검(확인 / 습지 수정 / 숨김)**하며, **습지명 검색과 날짜 기간 필터**로 원하는 이슈를 찾을 수 있는 무료 호스팅 웹서비스를 **2주 내 1차 완성**한다.

## Constraints
- **무조건 무료**: 호스팅, 지도, 뉴스 수집, DB, 자동 실행 스케줄 전부 무료 티어만 사용
- **기한**: 2주 내 1차 기능 실현
- **회원/로그인 없음**: URL만 알면 접속 (내부용, 접근 제한 없음)
- **뉴스 원문은 링크로만**: 제목·날짜·출처만 저장, 본문 저장/재게시 금지 (저작권)
- **습지 목록**: 습지센터 보유 목록 사용 — **파일 제공 필요** (좌표 포함 여부 확인 필요, 없으면 지오코딩 단계 추가)
- **팀 공유형 데이터**: 점검 상태·수정 내역이 모든 직원에게 공유 (개인별 저장 아님)
- **디자인**: PC+모바일 반응형 웹, 밝은 화이트 테마, 파랑 메인 색상, 토스풍 깔끔한 스타일
- **자동 수집 주기**: 매일 (구체 시각은 구현 시 결정)
- **작업 방식**: 최대한 자동으로, 비개발자 대상 쉬운 설명, 단계별 진행

## Acceptance Criteria
- [ ] 지도를 열면 습지센터 보유 목록의 습지들이 전국 지도에 표시된다
- [ ] 습지 관련 뉴스가 매일 자동으로 수집되어 해당 습지에 연결된다 (사람 개입 없이)
- [ ] 지도에서 습지별 이슈(뉴스) 건수를 한눈에 볼 수 있다 (대시보드 기능)
- [ ] 습지를 클릭하면 그 습지의 뉴스 목록(제목·날짜·출처·원문 링크)이 보인다
- [ ] 뉴스마다 점검 상태(미점검/확인됨)를 바꿀 수 있고, 다른 직원 화면에도 반영된다
- [ ] 잘못 연결된 뉴스는 다른 습지로 수정(재지정)할 수 있다
- [ ] 습지와 무관한 뉴스는 숨김 처리할 수 있고, 숨기면 지도에서 사라진다
- [ ] 습지명으로 검색해 해당 습지로 이동할 수 있다
- [ ] 날짜 기간 필터로 특정 기간의 이슈만 지도에 표시할 수 있다
- [ ] PC와 모바일 브라우저 모두에서 정상 동작한다
- [ ] 무료 호스팅에 배포되어 URL로 누구나 접속 가능하다

- **호스팅/DB**: Vercel/Netlify/Cloudflare Pages 무료 티어 + Supabase 등 무료 DB
- **핵심 의존성**: 습지센터 습지 목록 파일 (이름+좌표). 좌표 없으면 지오코딩 단계 필요
