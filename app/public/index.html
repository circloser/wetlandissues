    지도 설정

    지도 제공자는 지도 위 🗺 지도 메뉴에서 선택하세요. 무료 지도(OpenStreetMap)는 키가 필요 없습니다. 구글/네이버/카카오를 쓰려면 아래에 키를 입력하세요.

    VWorld: vworld.kr에서 무료 발급 후 도메인 등록

    카카오: developers.kakao.com JavaScript 키, 사이트 도메인 등록

    구글: console.cloud.google.com에서 Maps JavaScript API 활성화 + 결제 등록, 사이트 도메인 제한 권장

    네이버: NCP 콘솔 Maps 신규 Key ID(ncpKeyId) 발급 + 서비스 URL(도메인) 등록
